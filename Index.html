Text Formatting

There are three main types of text formatting. Those are bold, italics, and underling.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An inline element does not display its content on a new line. It keep the content on the same line as the previous content. Bold, italics and underline are all inline elements.

Images

An image element is an inline element which inserts pictures into the document. An image element is self-closing, which means it does not have a closing tag. it has one major attribute is self-closing, which mean it does not have closing tag. Ihas one major attribute, src, which is a filepath that points to the images' location. Those filepaths can either be absolute or relative.
