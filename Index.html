Text Formatting

There are three main types of text formatting. Those are bold, italics, and underling.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An inline element does not display its content on a new line. It keep the content on the same line as the previous content. Bold, italics and underline are all inline elements.

Vertical Text Spacing

There are two element that help with vertical spacing between block elements. A line break elements adds an emepty line and moves the content down to a line. Its similar to the ENTER key in a Google/word doc. A horizontal rule is a line break with a horizontal bar across the page. Both of these are block elements.
